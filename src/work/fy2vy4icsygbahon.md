# 问题
今天在折腾搭建vuepress博客，打算把语雀内容迁移到博客，发现语雀做了防盗链，导致图片无法显示。
# 如何解决呢？🤦‍
在网上找了一圈，试错了好几个小时，眼花腰疼终于找到了解决方案！🤩

😀 使用picGo图床工具+typora

picGo 是一个集成各大云存储的工具，省去了各种配置，一键上传云平台。
picGo 搭配 typora使用，操作md文档后，可直接右键上传云平台。再也不用担心图片失效问题啦！
