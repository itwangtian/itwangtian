# 个人博客定义

- 为什么搭建个人博客，自留地
- 写博客好处，利己利他，沉淀经验，帮助他人，扩大影响力，一次投入、持续产生复利，积累影响力。
- 什么时候写文章？**分享欲 **读到一篇高质量博客，有没有感叹到 什么时候我能写出来呀 ？用刚学新技术，实现了一个狂拽酷炫超吊的功能，你要不复盘总结记录下，岂不是太浪费啦！花半天解决的bug，防止下次不犯错，也帮助更多人解决问题
- 个人博客搭建方式 一键建站、静态博客、动手撸一个前后端手写
